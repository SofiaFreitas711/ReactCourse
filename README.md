## ðŸ“‹ Learning Notes

### What is React? 

React is a JS library for building UI.

React can create smooth transitions, and load content without the user feeling like the whole page is loading.

#### Why React?

In React, we define the target UI state, and React will figure out and perform the necessary steps to get to the goal.

### About React syntax

React code won't execute in the browser. For that to happen, it uses JSX (JavaScript Syntax eXtension) which is not a default JS feature

Component Function rules:
1. Function name must start with an upper case
2. Multi-word should be written in PascalCase

{} -> tells react it's about to create a dynamic part

Images should be used with import, for a better building deployment
